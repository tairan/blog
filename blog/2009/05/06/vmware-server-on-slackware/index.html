Install VMware Server 2.0 on Slackware 12.2

May 6th, 2009

在 Slackware 12. 2 上安装 VMware Server 2.0 后，总是提示 Login failed due to a bad username or password

解决办法： 到 http://linuxpackages.net/ 去下载 Linux-PAM 我用的是 Linux-PAM 1.0.4 i486

      # <strong>installpkg linux-pam-*.tgz</strong>

安装linux-pam后，重新安装 VMware Server 2.0 即可！
